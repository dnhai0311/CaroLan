Hướng dẫn sử dụng chương trình:
Cách 1: Vào folder caro/ , chạy CaroLan.sln và start project ( >= Visual Studio 2017 hoặc sử dụng cách 2 ) 
Cách 2: Vào folder caro/Server/bin/Debug/Server.exe để chạy frm Server (Cần chạy file này trước để khởi tạo server)
	Vào folder caro/Client/bin/Debug/Client.exe để chạy frm Client
